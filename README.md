# Sai Peram's Professional Portfolio

This repository contains a **personal portfolio website** built with **HTML and CSS** that showcases my professional experience in **Robotics, Autonomous Vehicles, Unmanned Drones, and Surgical Robotics**.

## About the Website

The website highlights my academic and research experience and is designed to provide a clean and visually appealing layout for professional presentation. Key technologies and concepts demonstrated include:

- **HTML** for structuring content, including sections for **About Me**, **Professional Experience**, and **Footer**.
- **CSS** for styling, including:
  - Custom fonts and color schemes for a professional look.
  - Box model properties for padding, margin, and borders.
  - Flexibility in layouts with centered content and responsive design.
  - Hover effects for links to improve user interactivity.
  - Highlighting key organizations and projects using distinctive colors and font weights.
- Semantic structuring of content for accessibility and readability.

## Professional Experience Showcased

- **ARTS Lab (Advanced Robotic Technologies for Surgery Lab)**  
  Researching AI-driven enhancements for surgical robotics to improve precision and safety.

- **NOVA: Self-Driving Research Lab**  
  Worked on autonomous vehicle navigation and multimodal sensor fusion systems, including **YOLOv8 for brake light detection** and **time-series forecasting for dynamic occupancy grids**.

- **Texas Aerial Robotics**  
  Developed swarm robotics solutions using **Multi-Agent Deep Deterministic Policy Gradient (MADDPG)** to safely control multiple drones in complex environments.
